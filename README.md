# 🧠 Mental Health Companion Chatbot

A safe, AI-driven chatbot designed to support student mental well-being.  
It detects user mood through **sentiment analysis** and generates **empathetic, motivational responses** along with **relaxation tips**.

---

## 🚀 Features
- Mood detection using **Hugging Face Transformers**
- Empathetic responses tailored to **Positive**, **Negative**, or **Neutral** moods
- Relaxation and motivational tips for students
- Simple and interactive **Streamlit UI**
